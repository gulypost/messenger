# Messenger

Личный проект мессенджера на C++.

Сейчас репозиторий содержит первые версии UDP-чата для Windows. Эти версии используются как начальный этап разработки сетевой части будущего полноценного мессенджера.

В дальнейшем проект будет расширяться: планируется добавить полноценный интерфейс, работу с пользователями, историю сообщений, передачу файлов и другие функции.

## Текущий этап разработки

На данный момент проект находится на ранней стадии.

Основная цель текущих версий — протестировать обмен сообщениями между компьютерами по протоколу **UDP** с использованием **Winsock2**.

## Последняя версия

Актуальная версия проекта: **`v0.2.0`**.

В этой версии программа одновременно принимает и отправляет сообщения, поэтому больше не нужно отдельно выбирать режим отправителя или слушателя.

Перейти к последней версии: [`v0.2.0`](v0.2.0/)

## План развития

В будущем планируется добавить:

* регистрацию и идентификацию пользователей;
* хранение истории сообщений;
* передачу файлов;
* сервер-посредник для связи между пользователями;
* графический интерфейс;
* поддержку разных операционных систем;
* более надёжный протокол обмена сообщениями.

## Требования для текущих версий

* Операционная система: **Windows**
* Компилятор: **MinGW g++** или **Visual Studio**
* Библиотека: **Winsock2**

## Быстрый запуск последней версии

Перейдите в папку `v0.2.0` и соберите проект:

```bash
g++ win_v0.2.0.cpp -o udp_chat.exe -lws2_32
```

После сборки запустите `udp_chat.exe` на двух компьютерах или в двух окнах консоли.

## История версий

### `v0.1.0`

Первая версия программы. Пользователь выбирает один из двух режимов:

* `sender` — отправка сообщений;
* `listener` — приём сообщений.

### `v0.2.0`

Обновлённая версия программы. Основные изменения:

* убрано разделение на `sender` и `listener`;
* добавлена одновременная отправка и приём сообщений;
* используются два потока;
* для завершения работы используется команда `/exit`.
